# Orbitani
[Orbitani Demo Page](https://orbitani.azurewebsites.net/)  

A web application locates the space debris orbiting the Earth in realtime. You can mark a place and see the exact number of debris passing by, and the trajectory.

## What exactly does it do?

Track and locate the space debris on a web app.

Find out the debris that will pass by the specific region.

Show the trajectory of each space debris.

Show the path of the moving debris within 2 hours (1 hour before from now and 1 hour after)

## Getting Started
### Installing Node.js
Please install node.js version v14.17.6 or higher  

Linux:
```sh
sudo apt install nodejs
```
Windows:  
Go to [Node.js download page](https://nodejs.org/en/download/) and download it.

### Building Project
```sh
npm install
```

### Runing Project
```sh
npm start
```

The project page will show on http://localhost:8080 or http://127.0.0.1:8080
